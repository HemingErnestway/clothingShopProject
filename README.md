# clothingShopProject

### Описание проекта
Интернет-магазин одежды, включающий женский, мужской и детский разделы. Товары разделены на множество категорий, а также имеют различные признаки для поиска, сортировки и фильтрации.

### Стак технологий
* HTML
* CSS
* JS
* Golang
* PostgreSQL
